Esse foi um projeto do primeiro semestre da faculdade de Sistemas de Informação na disciplina Desenvolvimento Web.
O objetivo era fazer um site usando linguagens Front End como Html, Css e Javascript, e um pouco de Back End, como Php.
Usei também banco de dados com MySql. 
Minha ideia foi fazer um site de compartilhamento de batidas e instrumentais para músicas, onde poderia ser feito um ca-
dastro para postar áudios e baixa-los ou compra-los.
